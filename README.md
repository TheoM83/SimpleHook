# SimpleHook


## Descripion
A simple WebHook written in Node to perform test on web pages.

## Install
        ### 1) Clone the Project
        ```
        git clone https://github.com/TheoM83/SimpleHook
        ```
        
        ### 2) Modify Parameters in Hook.js
        ```
        //Parameters
        const host = 'your.ip';
        const port = 4444;
        ```
        
        ### 3) Run the Script
        ```
        node Hook.js
        ```
